Metrics Frameworks

Frameworks for measuring success

HEART Framework

Google's HEART framework helps measure the quality of user experience:

  • Happiness: User satisfaction and attitudes
  • Engagement: Level of user involvement
  • Adoption: New user acquisition
  • Retention: User return rate
  • Task success: Efficiency and effectiveness

Pirate Metrics (AARRR)

Dave McClure's startup metrics for pirates:

  • Acquisition: How users find you
  • Activation: First happy experience
  • Retention: Repeated visits
  • Referral: User recommendations
  • Revenue: Monetization

North Star Framework

A single metric that best captures the core value your product delivers to customers:

  • Leading indicator of success
  • Measures customer value
  • Reflects current product strategy
  • Easy to understand and communicate
