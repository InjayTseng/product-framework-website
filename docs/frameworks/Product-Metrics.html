Product Metrics Frameworks

Measure what matters with proven product metrics frameworks

HEART Framework

Google's HEART framework helps measure user experience quality at scale, focusing on five key dimensions.

Framework Components

Happiness

User satisfaction and attitudes

  • User satisfaction scores
  • Net Promoter Score (NPS)
  • Customer feedback ratings

Engagement

User involvement and interaction

  • Session frequency
  • Session duration
  • Actions per session

Adoption

New user acquisition

  • New user signups
  • Feature adoption rate
  • Activation metrics

Retention

User return rate

  • Churn rate
  • Return user rate
  • Lifetime value

Task Success

User goal completion

  • Conversion rates
  • Task completion time
  • Error rates

AARRR (Pirate Metrics) Framework

The AARRR framework, also known as Pirate Metrics, focuses on the customer journey through five key stages.

Customer Journey Stages

Acquisition

How do users find you?

  • Traffic sources
  • Cost per acquisition
  • Landing page conversion

Activation

Do users have a great first experience?

  • Sign-up completion
  • Onboarding completion
  • First value moment

Retention

Do users come back?

  • Return rate
  • Engagement over time
  • Churn analysis

Referral

Do users tell others?

  • Referral rate
  • Viral coefficient
  • Share metrics

Revenue

How do you monetize?

  • Customer lifetime value
  • Revenue per user
  • Conversion to paid

North Star Framework

The North Star Framework helps teams align around a single, critical metric that best captures the core value of your product.

Framework Components

North Star Metric

The single metric that best captures value creation for users

Examples:
  • Spotify: Time spent listening
  • Airbnb: Nights booked
  • Facebook: Daily active users

Input Metrics

User Growth
  • New user acquisition
  • Activation rate
Engagement
  • Feature adoption
  • Session frequency
Network Effects
  • Viral coefficient
  • Content creation

Framework Comparison

Framework Best For Focus Area Complexity
HEART User experience measurement User satisfaction & behavior Medium
AARRR Growth tracking Customer journey & conversion Low
North Star Strategic alignment Value creation & growth Medium
